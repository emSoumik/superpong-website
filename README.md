## 🚀 **Quick Start: Get Playing in Minutes**

### 🎯 **Option 1: Browser Only (Super Easy)**
```bash
git clone https://github.com/emSoumik/SuperPong.git
cd SuperPong
npm install
npm run dev
```
**Open your browser, say "SuperPong", and start playing!**

### 🚀 **Option 2: Full Backend (Maximum Accuracy)**
```bash
# Terminal 1 - Frontend
npm run dev

# Terminal 2 - Backend with YOLO
pip install -r requirements.txt
python main.py
```
**Enhanced accuracy with Python backend processing.**
